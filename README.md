# Networking Lab - Basic LAN Setup

## Objective
To configure a basic LAN network using Cisco Packet Tracer and test connectivity between devices.

## Tools Used
Cisco Packet Tracer

## Topology
Two PCs connected to a switch using Ethernet cables.

## Configuration

PC0:
IP Address: 192.168.1.10  
Subnet Mask: 255.255.255.0  

PC1:
IP Address: 192.168.1.20  
Subnet Mask: 255.255.255.0  

## Connectivity Test
Ping test was performed from PC0 to PC1 and it was successful.
